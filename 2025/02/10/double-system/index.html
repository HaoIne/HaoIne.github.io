安装 windows 和 Kali Linux 双系统

之前一直使用在虚拟机上安装的 kali， 由于虚拟机的性能限制，使用起来一直很难受，便尝试安装了双系统，使用 kali
的 GRUB 引导在开机时选择系统。 第一次尝试双系统的安装，还是非常麻烦的，也遇到了挺多问题，在这里记录一下

首先，为 kali 系统分出一部分磁盘空间

分盘的话，可以使用 windows 自带的磁盘管理工具 diskmgmt.msc，但是由于 windows 的限制，在分盘时可分出的部分非常小，完全不够用。
而使用网上的一些磁盘管理工具软件又都要收费，我就使用了开源的 Gparted-live 镜像进行分盘

  1. 在 Gparted-live 官网下载 Gparted-live 镜像文件
  2. 在 rufus 官网下载 rufus 用于制作 Gparted-live 驱动U盘
  3. 在 windows 设置-系统-恢复 中找到立即重新启动，重启后进入 windows 系统恢复界面，选择疑难解答-从设备启动-removable device，便可进入 Gparted-live
  4. 在 Gparted-live 进行分盘操作

进行 Kali Linux 的安装

先在 Kali Linux 官网下载 linux 镜像,再使用 rufus 制作 Kali 驱动U盘，从U盘启动便可开始 Kali 的安装

安装时的问题

在安装的最后一步：结束安装程序时，我的安装程序一直卡在了运行 symple-cdd 那一步，后来了解之后发现这是用于联网安装 Kali 工具包的程序，由于官网在国外网速太慢，一直卡在这一步。只要在安装时不进行网络配置便可跳过这一步
